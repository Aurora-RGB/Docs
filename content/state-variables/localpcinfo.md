Information is mostly provided by LibreHardwareMonitor

On AMD Systems, CPU and GPU data is not available by default because of insecure driver used by LibreHardwareMonitor.
Go to Aurora Settings -> Plugin Management to enable it
- **Battery**
  - **ChargeStatus**
  - **LifePercent**
  - **PluggedIn**
  - **SecondsRemaining**
- **CPU**
  - **PowerUsage**
  - **Temperature**
  - **Usage**
- **GPU**
  - **FanRPM**
  - **PowerUsage**
  - **Temperature**
  - **Usage**
- **NET**
  - **DownloadSpeed**
  - **UploadSpeed**
  - **Usage**
- **NetworkAdapters**
- **RAM**
  - **Free**
  - **Total**
  - **Used**
- **Time**
  - **CurrentDay**
  - **CurrentHour**
  - **CurrentMillisecond**
  - **CurrentMinute**
  - **CurrentMonth**
  - **CurrentSecond**
  - **MillisecondsSinceEpoch**
